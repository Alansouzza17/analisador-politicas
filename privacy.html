Política de Privacidade

Este aplicativo (“Analisador Instagram”) utiliza a API do Instagram Basic Display exclusivamente para exibir informações autorizadas pelo usuário, como fotos, nome e dados públicos do perfil.

Coleta de Dados

O aplicativo coleta apenas dados fornecidos pelo próprio usuário ao realizar o login, como:

  • ID do Instagram
  • Nome de usuário
  • Legenda e URLs das fotos

Uso dos Dados

Os dados são utilizados apenas para análise e sugestões automáticas dentro do aplicativo.

Armazenamento

Nenhum dado é armazenado permanentemente. Todos os dados são apagados ao encerrar o uso.

Compartilhamento

Não compartilhamos nenhuma informação com terceiros.

Exclusão dos Dados

Para solicitar exclusão de dados, acesse: /delete
